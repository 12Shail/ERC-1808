# ERC-1808
ERC全称Ethereum Request for Comments，即以太坊开发者提交的协议提案，ERC后面的数字是提案编号。

ERC-1808是Cocos-BCX 1808在以太坊上的提案，本项目是基于Cocos-BCX 1808设计理念和提案的一个Demo实现。主要分四个部分：
* 链
* ERC-1808接口
* NFT组合资产
* 游戏示例
